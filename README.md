# FixPro

WordPress проект з Elementor.

## Структура проєкту

- `index.html` - головна сторінка
- `wp-content/` - WordPress контент (теми, плагіни, завантаження)
- `wp-includes/` - WordPress ядро файли
- `images/` - зображення проєкту

## Встановлення

Просто розгорніть файли на веб-сервері з підтримкою PHP та MySQL.
